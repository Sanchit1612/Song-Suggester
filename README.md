# Song Suggester

A full-stack web application that recommends songs based on the userâ€™s mood.  
The app fetches real-time music data using the iTunes Search API and displays song suggestions in a simple, responsive interface.

## Features
- Mood-based song suggestions (hype, sad, chill, etc.)
- Real-time data from the iTunes API
- RESTful backend using Express
- Clean and responsive frontend using HTML, CSS, and JavaScript

## Tech Stack
- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- iTunes Search API

## How it works
1. User selects or enters a mood.
2. The backend processes the request.
3. The server fetches matching songs from the iTunes API.
4. Songs are returned and displayed on the frontend.
